# Documentation

All project documentation lives here. The canonical research spec (`reqs.md`) and the implementation (`pilot/`) stay at the project root.

## Reading order

If you're reviewing in the morning and want the fastest path back into context:

1. **[00_start_here.md](00_start_here.md)** — TL;DR of where we are, what's the decision today, what I recommend. Read this first.
2. **[04_next_step_vision_ablation.md](04_next_step_vision_ablation.md)** — Detailed reasoning behind the proposed next experiment. Read this if you want to engage with the methodological argument.
3. **[05_open_questions.md](05_open_questions.md)** — Parking lot of decisions to revisit. Quick scan to see if any of these matter now.

If you need to refresh on what happened before:

4. **[02_journey.md](02_journey.md)** — Full chronological narrative from initial idea through Phase 1 completion (4,200 words; written by a research agent reading the actual artifacts).
5. **[03_methodology_log.md](03_methodology_log.md)** — Every methodological decision with the citation backing it. Includes Gate 1 results.
6. **[01_research_validation.md](01_research_validation.md)** — The literature audit that turned the original `reqs.md` into v2. Reference doc, not a sequential read.

## What each doc is for

| Doc | Purpose | Update frequency |
|---|---|---|
| 00_start_here.md | Morning briefing. Single source of truth for "what's next." | Rewritten each session |
| 01_research_validation.md | Literature audit + which citations survived. | Append-only |
| 02_journey.md | Chronological project narrative. | Append after each phase |
| 03_methodology_log.md | Decision log with citations + rationale. | Append after each Gate |
| 04_next_step_vision_ablation.md | Current next-step proposal in detail. | Replaced each new proposal |
| 05_open_questions.md | Flagged decisions to revisit. | Append as questions arise |

## Convention

- Numbered prefixes (`00_`, `01_`, …) reflect intended reading order, not chronology.
- Each doc has a one-line purpose statement at the top.
- All citations include arXiv IDs or DOIs.
- Anything labeled "Claude's note" is my opinion, separated from facts.
